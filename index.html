О сервере

Играем в режим выживания (с жизнями, без креатива). Делать можно что угодно. Есть моды, которые делают интереснее исследование в игре:

  • Оптимизация

  • Механизмы

  • Аэропланы

  • Деревни

  • Мини-карта

  • Фичи из новых версий

Версия: 1.21.1

Как начать играть

Кратко

  1. Установи Java

    java java

  2. Скачай SKLauncher и создай новую установку Neoforge 1.21.1

    SKLauncher SKLauncher

  3. Нажми ⋮ › Open Installation Directory (папка с игрой) и распакуй туда minecraft.zip

  4. Напиши мне, чтобы я добавил тебя в белый список и дал адрес сервера

    написать никите

  5. Запусти сборку, добавляй сервер в список мультиплеера и подключайся

Подробно

  1. Скачай и установи Java.
    На Windows должна устанавливаться вместе с лончером.

    java java

    На маке можно через brew: brew install openjdk@21

  2. Скачай SKLauncher для нужной платформы

    SKLauncher SKLauncher

  3. Запусти SKLauncher и перейди в оффлайн-режим

    P.S. Если есть лицензия майнкрафта, можно войти в аккаунт Microsoft

  4. Вводи никнейм и логинься

  5. Важно! Отправь никнейм мне, чтобы я добавил его в белый список сервера

    написать никите

  6. Нажми + и создай новую установку: 1.21.1 -> NeoForge

  7. Нажми ⋮ › Open Installation Directory (папка с игрой) и распакуй туда minecraft.zip

  8. Когда будет готово, запускай игру

    Нужно будет подождать загрузку файлов

  9. Выбирай язык и продолжай

    Советую поменять масштаб интерфейса в Options › Video Settings › Scale, чтобы всё не было таким гигантским

  10. Заходи в мультиплеер

    Соглашайся с тем, что серверами владеют люди

  11. Добавляй новый сервер

    Называй его как хочешь, впиши адрес. Узнай его лично

  12. Готово, подключайся!

    Здесь что-то может пойти не так: несовместимость модов, проблемы с подключением, отсутствие твоего никнейма в белом списке. Если что, пиши мне — разберёмся.

Как продожить играть

  1. Открыть SKLauncher

  2. Запустить сборку

  3. Перейти в мультиплеер

  4. Подключиться к серверу

Как поменять скин

  1. Зайди на сайт SKLauncher и войди в аккаунт

  2. В My Acccount › Customization укажи свой юзернейм из лончера

  3. Заходи в Skins и выбирай любой, который нравится

  4. Забирай себе по кнопке Assign skin to my account

  5. По такому же сценарию можно добавить себе плащ на странице Capes

  6. А ещё на странице Customization можно добавить скины и плащи из интернета вручную

Как включить ресурс-паки

Собрал для игры красивые ресурс-паки на анимации и листья, можно включить их в меню

  1. Зайди в Меню (Esc) › Options › Resource Packs

  2. Перенеси вправо Better-Leaves, FreshAnimations и Freshly Animated. Перенеси Default Dark Mode, если хочешь тёмный интерфейс

Почему не подключается

  1. Тебя нет в белом списке

    Напиши мне свой никнейм, чтобы я мог добавить тебя в правила

  2. Сервер упал

    Напиши мне — могу быстро проверить

  3. Провайдер

    Сервер в Германии, поэтому может пригодится VPN. Кстати, у моего друга есть отличный вариант

Почему тормозит на крутом ноуте

Если у тебя ноутбук на Windows, Minecraft может использовать встроенную видеокарту вместо дискретной. На это можно повлиять

Способ 1 — настройки лончера

  1. Открой SK Launcher

  2. Включи настройку Use Dedicated GPU / Использовать выделенную видеокарту

  3. Зайди в любой мир и проверь, что при нажатии на F3 можно увидеть твою видеокарту, а не Intel UHD, например

Способ 2 — настройки видеокарты

  1. Запусти майнкрафт из лончера

  2. Перейди в диспетчер задач и найди Java

  3. Нажми правой кнопкой и перейди к расположению файла

  4. Скопируй адрес в проводнике

  5. Перейди в настройки графики

  6. Добавь новое приложение

  7. Перейди по скопированному пути и выбери javaw.exe

  8. Найди OpenJDK Platform binary либо javaw.exe в списке

    Установи свою видеокарту как предпочтительную

  9. Перезапусти майнкрафт

  10. Зайди в любой мир и проверь, что при нажатии на F3 можно увидеть твою видеокарту, а не Intel UHD, например
